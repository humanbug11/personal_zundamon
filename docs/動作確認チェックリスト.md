# ずんだもんAIパートナー 動作確認チェックリスト

## 事前準備（確認済みなら✓）

- [ ] **VOICEVOX Engine** が起動している（http://127.0.0.1:50021/docs でAPIドキュメントが開ける）
- [ ] **.env** に `VITE_GEMINI_API_KEY` が設定されている
- [ ] `npm install` 済み
- [ ] `npm run dev` でアプリが起動し、ずんだもんのウィンドウが表示される

---

## 1. 画面・UI

| 確認項目 | 手順 | 期待結果 |
|----------|------|----------|
| ウィンドウ表示 | アプリ起動 | 透過ウィンドウにずんだもんが表示される |
| デフォルト表情 | 起動直後 | 通常（normal）の表情 |
| 操作パネル | 左上にマウスを乗せる | START ボタンと状態（IDLE）が表示される |
| ドラッグ移動 | ずんだもんをドラッグ | ウィンドウごと画面上で移動できる |

---

## 2. 会話開始

| 確認項目 | 手順 | 期待結果 |
|----------|------|----------|
| START 押下 | 左上の START をクリック | 状態が CONNECTING → READY に変わる |
| 初回挨拶 | START 後、数秒待つ | ずんだもんが先に挨拶する（音声＋口パク） |
| 音声再生 | 挨拶が流れる間 | ずんだもんの声（VOICEVOX）で聞こえる |
| 口パク | 話している間 | 口開き画像と口閉じ画像が音量に合わせて切り替わる |

---

## 3. 対話

| 確認項目 | 手順 | 期待結果 |
|----------|------|----------|
| 音声入力 | マイクに向かって話す | ずんだもんが反応して返答する |
| 画面共有 | 何か作業しながら話す | 画面内容に触れた返答が返ってくる |
| 感情タグ | 返答の内容に応じて | 表情が変わる（happy, smug, sad など） |
| 割り込み | ずんだもんが話している最中に話す | 再生が止まり、新しい会話に切り替わる |

---

## 4. 終了

| 確認項目 | 手順 | 期待結果 |
|----------|------|----------|
| STOP | 左上の STOP をクリック | 接続が切れ、状態が IDLE に戻る |
| アプリ終了 | ウィンドウを閉じる | アプリが終了する |

---

## トラブル時

- **START で ERROR になる**  
  - .env の API キー、VOICEVOX が 50021 で起動しているか確認
- **音声が出ない**  
  - VOICEVOX Engine が起動しているか、http://127.0.0.1:50021/speakers が 200 で返るか確認
- **マイクが効かない**  
  - ブラウザ/OS のマイク許可、Electron の権限を確認
